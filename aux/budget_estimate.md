# HydraR ISC Grant Budget Estimate

**Total Requested: $14,400 USD**

## Funding Breakdown
Based on a recent conversion rate of 1 USD = 1.39 AUD, the requested grant of $14,400 USD provides $20,016 AUD. 
With an analyst rate of $200 AUD per hour, this budget covers approximately **100 hours** of focused development time.

## Allocation of Hours (100 hours total)

### 1. Integration and Infrastructure (40 hours)
- Dynamic Network Repair and Network Plasticity.
- Native Restart checkpointing and state-hashing of output files.
- Persistent memory integration (DuckDB).
- Docker and Git Worktree integration.
- API Key Security via SSH agents.

### 2. Ecosystem Interoperability (40 hours)
- Bioconductor and CRAN node integration.
- Nextflow interoperability.
- Singularity / Apptainer sandbox environments.
- Integration with ellmer, mall, targets, and native Reticulate support.

### 3. Community and Hardening (20 hours)
- Package hardening, unit testing.
- Agents and Skills management lifecycle.
- Documentation and vignettes.

### Financial Stewardship
Funds will be managed by **APAF Bioinformatics** (Macquarie University). Resources are allocated directly to **Research Engineering time** (Ignatius Pang & Aidan Tay) to rapidly mature the HydraR agentic architecture.
